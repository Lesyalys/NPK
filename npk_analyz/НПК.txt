Я хочу сделать проект по проведению экзаменов. 
Платформа на которой преподаватель может создавать свои экзамены а студенты проходить их
пока без кода, обсудим архитектуру, инструменты и лучшие практики

стек фронт - js, html, css (Что бы не зависеть или минимально завесить от сторонних Фреймворков, библиотек итд)
стек бек - nodejs, express 

хранение - реляционная база данных и json ( думаю о supabase или firebase, можно иное по лучшим практикам и меньшим затратам)
важно - Хранить минимум или вообще не хранить чувствительных данных (поролей, почты, телефона итд)

базово
функционал преподавателя: 
создание учетной записи преподавателя (регистрация по почте и/или иные менее затратные методы)
создание учетной записи студента (заводит преподаватель, например, по номерам зачетных книжек и ФИО)
создание / редактирование экзамена (хранить в json просто структуру вопросов для экзамена с типом полей - открытый вопрос, выбор варианта, выбор нескольких вариантов, поставить в правильной последовательности итд. Имя экзамена, его видимостью для студентов итд)


функционал студента:
Зайти в свою учетную запись по данным, выданным преподавателем
Пройти видимые экзамены

[Клиент (JS, HTML, CSS)] <--> [API Gateway / Node.js (Express)] <--> [PostgreSQL (данные)]
                                                                  |
                                                                  +--> [JSON-файлы (структуры экзаменов)]
                                                                  |
                                                                  [Redis (опционально, для сессий)]
